# Weather App

## Live Demo

🌐 https://psychiatry127.github.io/weather-app/

---

## Description

A simple and modern weather application that allows users to search for any city and view real-time weather conditions, air quality information, and local time through a clean and responsive interface.

---

## Features

- Search for weather information by city name  
- Detect weather using user’s current location  
- Real-time temperature, wind, humidity, pressure, and visibility data  
- Air quality information including PM2.5, PM10, CO, and NO₂  
- Live local time display for selected locations  
- Light and dark mode support  
- Responsive and clean user interface  

---

## Technologies Used

- HTML  
- CSS  
- JavaScript  
- OpenWeather API  

---

## How to Run

- Download or clone the repository  
- Open `index.html` in any modern web browser  
- Start searching for cities or use the location feature  

No installation or build process is required.

---

## API Information

This project uses the OpenWeather API to retrieve weather and air quality data. The API key is used directly in the frontend for simplicity. In production environments, API keys should be secured using a backend service.
